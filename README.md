## 🌌 Tentang Repository
Selamat datang di pusat distribusi plugin resmi **INF Project**. Repository ini berisi kumpulan *script* dan *plugin* WhatsApp Bot berformat **ECMAScript Modules (ESM)** yang modern, ringan, dan siap pakai. 

Semua kode di sini dirancang dengan struktur *clean code* dan antarmuka *cyber-modern* ala **Erine-AI**, cocok untuk para *developer* yang ingin fitur botnya tampil beda dan minim *error*.

## ✨ Fitur Utama
- **ESM Ready:** Struktur kode modern menggunakan `import/export`.
- **Plug & Play:** Mudah dipasang di berbagai *base* bot MD.
- **Auto-React & Rich UI:** Dilengkapi dengan respons visual yang rapi dan interaktif.
- **High Performance:** Kode yang dioptimasi untuk menghindari beban memori (RAM) berlebih.

## 🛠️ Cara Instalasi
1. Pilih *plugin* yang ingin kamu gunakan di dalam folder repo ini.
2. Salin kode (Raw) atau unduh file `.js` tersebut.
3. Masukkan file ke dalam folder `plugins` di dalam *base* bot kamu.
4. Sesuaikan `apikey` atau nama modul jika diminta di dalam file tersebut.
5. *Restart* bot (atau gunakan fitur *auto-reload* jika bot kamu mendukungnya).
